# Changelog

## 0.2.0

- Adds macOS/Linux and Windows one-command installers
- Uploads VSIX assets to GitHub Releases for friendlier distribution
- Preserves original filenames on download and avoids overwriting with numeric suffixes
- Uses macOS Downloads by default and temp directories on Windows and Linux
- Tries to open explicitly in Chrome across macOS, Windows, and Linux with fallback to the default browser

## 0.1.0

- Initial release
- Adds an Explorer command to download remote HTML files locally and open them in Google Chrome
